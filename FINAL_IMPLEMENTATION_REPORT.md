### 6. Agent Behavior Modes

**Four Configurable Modes:**

1. **Strict Compliance** - Production systems
   - Zero tolerance for violations
   - All governance enforced
   - Human review required

2. **Guided Development** - Default mode
   - Balance governance and velocity
   - Critical violations blocked
   - Warnings allowed

3. **Exploratory** - Prototyping
   - Minimal governance
   - Fast iteration
   - Must validate before merge

4. **Emergency Override** - Incidents
   - All checks disabled
   - Full audit trail
   - Incident ID required
   - 48-hour maximum
   - Post-mortem required

**Mode Switching:**
```json5

  // ... mode definitions
}
```

## Philosophical Foundation

### The Aletheia Principle (Core Philosophy)

**Mathematical Form:**
```
δ∫C(t) dt = 0
```

**Interpretation:**
"The variation of concealment integrated over time must equal zero at equilibrium."

**Meaning:**
Systems naturally evolve toward transparency. Information wants to be explicit. The path of least resistance is truth.

**Implications:**
- Documentation is not overhead—it's destiny
- Complexity that cannot be explained should not exist
- Hidden knowledge is technical debt
- Transparency is the default; concealment requires justification

### The Generalized Variational Principle (GVP)

**Mathematical Form:**
```
δS[T] = 0
```

**Interpretation:**
"The variation of the action functional over all trajectories is zero."

**Meaning:**
Like light taking the path of least time, software evolves along paths of minimal action. Coherent systems are more stable. Consistency emerges naturally.

**Implications:**
- Fight against inconsistency is fighting against nature
- Establish coherent patterns; the system will maintain them
- Refactoring toward coherence is always worth it
- Technical debt is action accumulated above the minimum

### The Integration: Aletheia + GVP

**Combined Principle:**
```
Systems evolve to minimize both concealment and action,
achieving maximum transparency with maximum coherence.
```

**Result:** The ideal software system is:
1. **Transparent** (Aletheia): Everything is explicit and documented
2. **Coherent** (GVP): Everything is consistent and minimal
3. **Preserved** (CPS): Nothing is lost, everything is traceable
4. **Grounded** (Ontology): All concepts have clear existence criteria

This is the philosophical foundation of MGFTS.
